/**** Orden de ejecucion ****/

servidor.py

index.html

/**** Ejecucion ****/

python nombre.py

abrir index.html en el navegador

/**** Intrucciones ****/

Primero se debe ejecutar el servidor.py para esto de tiene que
tener instalado websocket

Para instalar websocket ejecute este comando: pip install websockets

Despues de ejecutar el servidor se tiene que abrir index.html
y jugar


NOTA: La palabra a encontrar es "pudin", si se quiere cambiar la palabra
abrir el archivo cliente.js y en la variable palabra modificar a la que se
desea.
